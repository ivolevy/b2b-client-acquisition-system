   Sistema transformado de búsqueda de propiedades por zona a
   SISTEMA B2B DE CAPTACIÓN DE CLIENTES POR RUBRO EMPRESARIAL

 SISTEMA DE VALIDACIÓN AUTOMÁTICA:

   EMAILS:
    Formato válido (RFC 5322)
    Filtra falsos (example.com, test.com, noreply@)
    Prioriza corporativos (contacto@, info@, ventas@)

   TELÉFONOS:
    Entre 7-15 dígitos
    Formatos internacionales (+34, +52, etc.)
    Filtra falsos (000000, 111111, 123456)

   CRITERIO DE VALIDACIÓN:
    Empresa válida = Nombre + (Email válido O Teléfono válido)

 FLUJO DE TRABAJO:

   1. Usuario selecciona RUBRO (ej: tecnologia)
             ↓
   2. Sistema busca en OpenStreetMap
             ↓
   3. Opcionalmente scrapea sitios web (respeta robots.txt)
             ↓
   4. VALIDA emails y teléfonos automáticamente
             ↓
   5. Filtra solo empresas VÁLIDAS (con contacto verificado)
             ↓
   6. Guarda en SQLite con campos de validación
             ↓
   7. Exporta a CSV/JSON estructurado
             ↓
   8. Usuario obtiene DATOS LISTOS PARA CRM

 CARACTERÍSTICAS PRINCIPALES:

    Búsqueda por rubro empresarial (no por zona geográfica)
    10 rubros predefinidos + fácil de expandir
    Validación automática de emails y teléfonos
    Web scraping ético B2B (respeta robots.txt)
    Solo datos con contacto válido (email O teléfono)
    Base de datos SQLite optimizada
    Exportación estructurada CSV + JSON
    API REST completa con Swagger
    Scripts interactivos sin programar
    Estadísticas en tiempo real
    Prevención de duplicados
    100% gratuito (OpenStreetMap)
